AR Language Translator

A modern interactive language translation web app built with HTML, CSS, and JavaScript on CodePen Project.
This project demonstrates front-end UI design combined with language switching/translation logic and real-time user interaction.

Features
🈯 Multi-language translation interface (AR + multilingual support)
⚡ Instant text translation UI interaction
🎯 Clean, minimal and user-focused design
📱 Fully responsive layout (mobile + desktop)
🔄 Dynamic language switching system
🧠 JavaScript-powered translation logic
🎨 Modern UI styling with smooth visual hierarchy
🛠️ Built With
HTML5
CSS3
JavaScript (Vanilla JS)
DOM manipulation
Responsive Web Design
CodePen
